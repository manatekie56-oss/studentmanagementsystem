# 🎓 Student Management System

## 📖 About

Student Management System is a Java-based console application developed using Object-Oriented Programming (OOP) concepts and JDBC database connectivity. The system allows users to manage student records efficiently by performing operations such as registration, searching, updating, deleting, and storing student data in a MySQL database.

---

## ✨ Features

* Register Student
* Display All Students
* Search Student by ID
* Update Student Information
* Delete Student
* Save Student to MySQL Database
* Display Students from Database
* Input Validation
* Custom Exception Handling
